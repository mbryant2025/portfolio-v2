Overview

SLogo is a Java application that is an IDE and viewer for a custom implementation of the Logo programming language. Users can control multiple turtles and run programs that can draw shapes, move the turtles, and more.

Role

This project was completed as part of a team of 4 students for CS308 at Duke University. I was responsible for the graphics controller, rendering the Animals, and the functionality of the frontent elements.
