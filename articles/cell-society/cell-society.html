Overview

Cell Society is a Java application that can run various cellular automata simulations such as Conway's Game of Life, Spreading of Fire, and Predator-Prey. The software comes with various ways to view and adjust the simulation in real time.

Role

This project was completed as part of a team of 3 students for CS308 at Duke University. I was responsible for the entire backend of the project and for rendering the simulation in JavaFX.
