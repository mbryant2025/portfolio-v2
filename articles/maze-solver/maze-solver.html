Overview

GUI to make mazes that will be auto-solved. Click below to run the simulator!

Features

The algorithm solves using A* and is implemented in TypeScript. The GUI is implemented in React. You can update the walls, start, and end point, and the algorithm will automatically re-solve the maze.

Bonus features include enabling/disabling diagonal movement, clicking and dragging to draw walls, a template maze, and a button to clear the maze.

History

This project was initially implemented in Python with a GUI made in Tkinter in 2021. With the intent of embedding it into this website, I decided to re-implement it in TypeScript in 2023.

Also, if it wasn't evident from the screenshot, the old version was not perfect! Upon re-implementing I was sure to remove these bugs.
